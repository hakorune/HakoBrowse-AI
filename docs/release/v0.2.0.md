# HakoBrowseAI v0.2.0

## Release Notes (JP)
今回のリリースでは、Function Calling とスキル運用の実用性を大きく強化しました。

### Highlights
- `http_request` を正式ツール化
  - メソッド/ヘッダー/ボディ/タイムアウト/リダイレクト対応
  - `auth_profile` 指定でAPIキーを自動注入
- Tool API Profiles を追加
  - 設定画面から外部APIキーを登録・編集
  - `allowedHosts` で送信先ホスト制限
- スキル運用を改善
  - `load_skill_index` / `load_skill` / `load_skill_file` の段階読込
  - スキル画面で不足 `auth_profile` を警告表示
  - デフォルトスキルを同梱（`Weather Check` ON、`Moltbook Post` OFF）
- デバッグ/運用改善
  - `--default-state` 起動（保存状態を読まないプレビュー）
  - API未設定でもUI起動し、後から設定可能
  - セッション保存の安定化（イベント駆動 + フォールバック）

### Notes
- スキルの外部編集後は `/reload_skill` を実行してください。
- 認証付き `http_request` は `auth_profile` を明示してください。

## GitHub Release Text (Short)
